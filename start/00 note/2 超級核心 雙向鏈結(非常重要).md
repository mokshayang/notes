### 直接打上兩個 [[ 中括號

會出現已經有的筆記項目

### 後面接#可以選擇筆記段落

[[1 Markdown 語法#範例 :]]
[[]]

### 後面接#^可以選擇筆記*行數*

[[1 Markdown 語法#^884276]]

[[1 Markdown 語法#^601365]]


### 後面接#^*編輯* 可以選擇筆記指定行數

* 必須在鏈結文本中的行數做編輯

* 自定義鏈結名稱 :
* 再 " #^*編輯* " 之後 打上 " | "然後接名稱 [[1 Markdown 語法#^601365|測試]]


###  鏈結 展現文本內容

* 上述的功能 在開始前先打上 ' ! ' 之後在接上 " [[  "
*

* 
*  '#' 選擇段落
* 
* ^選擇行數(可編輯)
*
