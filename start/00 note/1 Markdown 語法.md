簡單的文字排版語法

法則 : 打完一行空一行 ^601365

這樣+上 markdown 排版，好看

## 語法

### 標題 :

### '# ' : 標題大小

* 一共有六個 ^8842
	* '# (空一格)' : 最大
	* '###### (空一格)' : 最小
	* 
### '# ' : #標題大小

	* '(空一格)#(直接輸出內容)' : 變成標籤

### ' * ' : 斜體

* 前後各+一個 *
* 或選取文字 crtl + I

### ' ** ' : 粗體

* 前後各+兩個 **
* * 或選取文字 crtl + B

#### 註 : 若只在行尾變化，只要打前面的，然後換行

* ' * ' 開始斜體。斜體結束必須在打上  ' * '，若直接換上，則不需要結束(換行等於結束)
* 
* '#' : 若打在行尾處 會等於 標籤 (標籤可在標籤檢視中查看所有文本的標籤)
### ' ~~ ' : 刪除線 :

* ~~這是一個測試~~ OK
* 

## 列表 :

### 有序列表 : 

___
打上數字+上一個 .  空一個  ，然後輸出內容 
Enter 後 會接續數字
___

### 無序列表 : 

___
打上' * '+上一個 .  空一個  ，然後輸出內容 
Enter 後 繼續無序列表
___

### 列表可用 TAB 鍵，產出子列表

#### 範例 :

1. 這是一個列表
2. 天氣
	1. 晴天
	2. 有風
